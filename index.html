Federated Learning with MNIST and CIFAR-10

This repository contains an implementation of a Federated Learning (FL) system using the Flower framework, applied to the MNIST dataset. The code simulates a federated learning system where multiple clients collaboratively train a global model without sharing their local data.
Federated Learning

Virtual Chat using TCP Sockets

This project implements a fully functional multi-client chat system using Python and socket programming, enabling real-time communication between multiple users on a local network. The application supports global and private messaging, file transfer, automatic server discovery, and optional integration with an artificial intelligence bot, practically demonstrating networking concepts, concurrency, communication protocols, and client-server architecture.
Network Computing

Low-pass FIR filter

Digital Signal Processing project that implements a low-pass FIR filter in Python for noise removal in audio signals. The system performs frequency-domain analysis using the Fourier Transform, applies Hamming windowing, and adjusts filter parameters for different voice profiles, practically demonstrating concepts of DSP, digital filtering, and spectral analysis.
DSP | FIR Filter | Signal Processing

Image and Video Quality Reductor

Command-line tool developed in Python for multimedia processing, capable of resizing images, compressing videos, and applying compression or effects to audio using FFmpeg. The project demonstrates pixel manipulation, frame processing, multimedia pipeline automation, and integration with libraries such as OpenCV, PIL, and SciPy.
Digital Image Processing
